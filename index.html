Life is an puzzle game I'm actively developing for Android. It's a neat little game based on Conway's Game of Life, and is on the Google Play Store for beta testing right now.

Dodger.io

Web

Dodger.io is a Javascript game that I'm developing with my roommate. It's a classic endless runner game with a twist, multiplayer!


The game is using the HTML5 Canvas element for client-side rendering, with a websocket connection to our Node.js server. (Hosted with OpenShift)


For this project I also built a backend admin analysis platform with Javascript and jQuery. It accesses the games internal API's to retrieve data and statistics from the MongoDB database.



MyGizmo

Android

A project I worked on while working at Trayak. MyGizmo is a mobile application for Android and iOS that lets you manage and trade your smartphones, and other devices. Some of the technologies I worked on for this project include: native Android development, Java server implementation with MySQL database (Hibernate), Javascript (jQuery), NodeJS.
