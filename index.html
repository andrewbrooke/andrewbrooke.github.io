Dodger.io

Dodger.io is a Javascript game that I'm developing with my roommate. It's a fun little endless runner game with a twist, multiplayer!

The game is using the HTML5 Canvas element for client-side rendering, with a websocket connection to our Node.js server.

For this project I also built a backend admin analysis platform with Javascript and jQuery. It accesses the games internal API's to retrieve data and statistics from the MongoDB database.
